MateCart Power BI Project
Overview

Power BI project analyzing MateCart’s e-commerce data. The project starts from an Excel dataset and includes dashboards for payments, orders, shopping carts, and deliveries.

Features

Cleaned and transformed data in Power Query.

Calculated metrics using DAX (total payments, discounts, product quantities).

Interactive dashboards with slicers for year, quarter, and location.

Visualizations follow MateCart’s brand identity for clarity and consistency.

Data

Source: Excel datasets (Orders, Users, Payments, Products, Carts).

Tables in Power BI: Users, Orders, Shopping Cart, Payments, Inventory Movement, Calendar, Measures.

ER Model

Shows updated table structure and relationships for proper analysis and metric calculation.

Future Work

Optimize payment methods and cart conversion.

Improve order tracking and dashboard metrics.
